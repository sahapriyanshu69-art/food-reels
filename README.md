# Food Reel App

A full-stack MERN food reel platform.

## Features
- User authentication
- Food partner authentication
- Upload food videos
- Like / Save reels
- Partner profile
- Vertical reel feed

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- ImageKit
- JWT Authentication

## Run Locally

Backend:
npm install
npx nodemon server.js

Frontend:
npm install
npm run dev
